# 6. Mathematical Reasoning About Equations

## Understanding Equation Manipulation

Being able to derive relationships from given equations.

---

## Basic Example

```
Given: a + b = b + c

Question: What can we conclude about a and c?

Solution:
a + b = b + c
a + b - b = b + c - b    (subtract b from both sides)
a = c

Visual:
    a + b = b + c
    └─┬─┘   └─┬─┘
      │       │
      └───┬───┘
       same value
    
    Therefore: a = c
```

```cpp
#include <iostream>
using namespace std;

int main() {
    int b = 5;
    int sum = 12;  // a + b = 12 and b + c = 12
    
    // From a + b = 12
    int a = sum - b;  // a = 12 - 5 = 7
    
    // From b + c = 12
    int c = sum - b;  // c = 12 - 5 = 7
    
    cout << "a = " << a << ", c = " << c << "\n";
    cout << "Therefore a = c: " << (a == c ? "True" : "False") << "\n";
    
    return 0;
}
```

## Chain of Equations

```
Given: x + y = y + z = z + w

This means:
1. x + y = y + z  →  x = z
2. y + z = z + w  →  y = w

Therefore: x = z and y = w

Visual:
Position:  1   2   3   4
Value:     x   y   z   w
           └───┬───┘
            x = z
               └───┬───┘
                y = w

Pattern: x y x y (alternating)
```

```cpp
#include <iostream>
using namespace std;

int main() {
    int sum = 10;
    
    // If x + y = y + z = z + w = 10
    // Then x = z and y = w
    
    int y = 3;  // Choose y
    int x = sum - y;  // x = 10 - 3 = 7
    
    int z = x;  // From x = z
    int w = y;  // From y = w
    
    cout << "Values: " << x << " " << y << " " << z << " " << w << "\n";
    cout << "Check: " << (x+y) << " = " << (y+z) << " = " << (z+w) << "\n";
    
    return 0;
}
```

## Difference Equations

```
Given: |a - b| = maximum
Also: a + b = S (constant sum)

To maximize |a - b|, make a and b as far apart as possible.

If a + b = 10:
- Try a=0, b=10: |0-10| = 10 ✓ (maximum)
- Try a=1, b=9:  |1-9|  = 8
- Try a=5, b=5:  |5-5|  = 0 (minimum)

Pattern: One should be minimum, other maximum
```

```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    int sum = 20;
    int min_val = 0;
    int max_val = 20;
    
    // For maximum difference with sum constraint
    int a = min_val;
    int b = sum - a;
    
    cout << "a = " << a << ", b = " << b << "\n";
    cout << "Sum: " << (a + b) << "\n";
    cout << "Difference: " << abs(a - b) << "\n";
    
    return 0;
}
```

## System of Linear Equations

```
Given:
a + b = 10
a - b = 2

Solve:
From equation 2: a = b + 2
Substitute in equation 1: (b + 2) + b = 10
                          2b + 2 = 10
                          2b = 8
                          b = 4

Then: a = b + 2 = 4 + 2 = 6

Check: 6 + 4 = 10 ✓
       6 - 4 = 2  ✓
```

```cpp
#include <iostream>
using namespace std;

int main() {
    // Solve: a + b = 10, a - b = 2
    
    int sum = 10;
    int diff = 2;
    
    // Formula: a = (sum + diff) / 2
    //          b = (sum - diff) / 2
    
    int a = (sum + diff) / 2;  // (10 + 2) / 2 = 6
    int b = (sum - diff) / 2;  // (10 - 2) / 2 = 4
    
    cout << "a = " << a << ", b = " << b << "\n";
    cout << "Check sum: " << (a + b) << "\n";
    cout << "Check diff: " << (a - b) << "\n";
    
    return 0;
}
```

## Bounds and Constraints

```
Given: 0 ≤ a ≤ N
       0 ≤ b ≤ N
       a + b = S

Question: What's the valid range for a?

Analysis:
Minimum a: When b is maximum
  b ≤ N, so minimum a = S - N (if S > N)
  But a ≥ 0, so minimum a = max(0, S - N)

Maximum a: When b is minimum
  b ≥ 0, so maximum a = S
  But a ≤ N, so maximum a = min(S, N)

Range: max(0, S - N) ≤ a ≤ min(S, N)
```

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int N = 10;
    int S = 15;
    
    // Find valid range for a
    int min_a = max(0, S - N);
    int max_a = min(S, N);
    
    cout << "Valid range for a: [" << min_a << ", " << max_a << "]\n";
    
    // Example: If a = 7
    int a = 7;
    int b = S - a;  // b = 15 - 7 = 8
    
    bool valid = (a >= 0 && a <= N && b >= 0 && b <= N);
    cout << "a=" << a << ", b=" << b << " is valid: " << (valid ? "Yes" : "No") << "\n";
    
    return 0;
}
```

## Maximization with Constraints

```
Maximize: f(a, b) = |a - b|
Subject to: a + b = S
            0 ≤ a ≤ N
            0 ≤ b ≤ N

Solution approach:
Since a + b = S (constant), we have b = S - a

|a - b| = |a - (S - a)| = |2a - S|

This is maximized when a is as far from S/2 as possible.

Cases:
1. If S ≤ N: Make a = 0 or a = S (one extreme)
2. If S > N: Make a = max or a = min within bounds
```

```cpp
#include <iostream>
#include <algorithm>
#include <cmath>
using namespace std;

int main() {
    int N = 5;
    int S = 7;
    
    // Find a that maximizes |a - b| where a + b = S
    
    int min_a = max(0, S - N);
    int max_a = min(S, N);
    
    // Try both extremes
    int a1 = min_a;
    int b1 = S - a1;
    int diff1 = abs(a1 - b1);
    
    int a2 = max_a;
    int b2 = S - a2;
    int diff2 = abs(a2 - b2);
    
    int max_diff = max(diff1, diff2);
    
    cout << "Maximum |a - b| = " << max_diff << "\n";
    
    return 0;
}
```

## Visual: Equation Transformation

```
Original: a + b = b + c

Step 1: Subtract b from both sides
        a + b - b = b + c - b
        
Step 2: Simplify
        a = c

Geometric view:
        
    [a][b]  =  [b][c]
     └──┬──┘    └──┬──┘
      same       same
       sum        sum
    
    Remove [b] from both:
    [a] = [c]
```
